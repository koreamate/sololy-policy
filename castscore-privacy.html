3. 카메라 및 AI 데이터 처리

CastScore는 다음 두 가지 목적으로 카메라를 사용합니다.

3-1. 악보 촬영 및 AI 분석

  • 사용자가 직접 악보를 사진으로 찍어 앱에 추가할 수 있습니다.
  • 촬영된 이미지는 사용자 본인 계정의 클라우드 저장소(Supabase Storage)에만 저장됩니다.
  • AI 가사 추출 기능 사용 시, 해당 이미지가 Anthropic Claude API로 전송될 수 있습니다 (사용자가 해당 기능을 실행한 경우에만).
  • 사용자가 악보를 삭제하면 이미지도 함께 삭제됩니다.

AI 데이터 처리에 관한 추가 안내:

  • 전송된 이미지는 Anthropic의 모델 학습에 사용되지 않습니다 (Claude API 기본 정책).
  • 처리 완료 후 Anthropic 측에서 30일 내 자동 삭제됩니다.
  • 사용자가 AI 기능(가사 추출, 메타데이터 분석)을 사용하지 않으면 외부 전송이 발생하지 않습니다.
  • Anthropic의 개인정보 처리에 대한 상세 내용은 Anthropic Privacy Policy를 참고하세요.

3-2. 고개 기울기 페이지 넘김 (핸즈프리 모드)

  • 전면 카메라로 사용자 얼굴을 인식해 고개 움직임(기울기 각도)을 감지하여 페이지를 자동으로 넘깁니다.
  • 모든 처리는 기기 내에서만 이루어집니다 (온디바이스 처리, Google ML Kit Face Detection).
  • 얼굴 이미지나 특징점 데이터는 저장되지 않으며 외부로 전송되지 않습니다.
  • 기능 사용 중에만 카메라가 활성화되며, 기능 종료 시 즉시 중단됩니다.
  • 사용자가 언제든 비활성화할 수 있습니다.

3. Camera and AI Data Processing

CastScore uses the camera for two purposes:

3-1. Sheet Music Capture and AI Analysis

  • Users can photograph sheet music to add to the app.
  • Captured images are stored only in the user's own cloud storage (Supabase Storage).
  • When using AI lyrics extraction, images may be sent to the Anthropic Claude API (only when the user initiates this feature).
  • Images are deleted when the user deletes the score.

Additional Information on AI Data Processing:

  • Submitted images are not used for Anthropic's model training (Claude API default policy).
  • After processing, images are automatically deleted by Anthropic within 30 days.
  • If the user does not use AI features (lyrics extraction, metadata analysis), no data is transmitted externally.
  • For details on Anthropic's data handling, see the Anthropic Privacy Policy.

3-2. Head Tilt Page Turning (Hands-Free Mode)

  • The front camera detects head tilt angle using face recognition to automatically turn pages.
  • All processing occurs entirely on-device (Google ML Kit Face Detection).
  • No facial images or landmark data are stored or transmitted externally.
  • The camera is active only while the feature is in use and stops immediately when disabled.
  • Users can disable this feature at any time.
